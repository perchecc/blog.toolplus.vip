# windows电脑快速实现对手机抓包-安卓篇

### 一、基础环境配置（Charles 端）

无论目标设备是什么 Android 版本，首先完成电脑端 Charles 的基础配置：

1. **安装 Charles 根证书到电脑**  
   `Help` → `SSL Proxying` → `Install Charles Root Certificate` → 安装到「受信任的根证书颁发机构」

2. **启用 SSL 代理**  
   `Proxy` → `SSL Proxying Settings` → 勾选 **Enable SSL Proxying**  
   添加规则：Host 填 `*`，Port 填 `443`（或 `*` 捕获全部端口）

3. **查看代理地址**  
   `Help` → `Local IP Address` 查看电脑 IP，Charles 默认端口为 `8888`

---

### 二、Android设备

如果设备系统低于 Android 7.0，只需安装证书到**用户目录**即可：

1. 手机与电脑同 WiFi，设置代理为电脑 IP + 8888
2. 手机浏览器访问 `chls.pro/ssl`，下载并安装证书
3. Charles 弹出 Allow 请求时点击允许，即可抓取 HTTPS（包括 WebView）
